# CV-AYUNDA-PERMATA-SEJAHTERA
CV Ayunda Permata Sejahtera adalah perusahaan manufaktur hasil tembakau yang berdedikasi tinggi dalam menghasilkan produk rokok olahan berkualitas. Berbasis pada pemilihan bahan baku daun tembakau dan cengkeh terbaik dari para petani lokal, kami menjaga cita rasa autentik dan kepuasan konsumen secara berkelanjutan.
